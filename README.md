# UK Road Safety Accident Analysis using Spark MLlib

## Project Overview
This project analyzes UK road safety accident data using Apache Spark and Scala. The objective is to predict accident severity using machine learning models based on road, environmental, and temporal conditions.

---

## Dataset
Dataset: UK Road Safety Accidents Dataset

Processed dataset used in the ML phase:
Final_data (2).csv

---

## Technologies Used
- Apache Spark
- Scala
- Spark MLlib
- VS Code
- Java JDK

---

## Spark Version
Apache Spark 3.x

---

## Environment Setup
Requirements:
- Java JDK 8 or later
- Scala 2.12
- Apache Spark 3.x
- VS Code or IntelliJ IDEA

---

## How to Run the Project

### Start Spark Shell

```bash
spark-shell
```

### Load Scala File

```scala
:load MLOperationsImproved.scala
```

### Run the Program

```scala
MLOperationsImproved.main(Array())
```
